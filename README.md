## 🏪项目介绍
致`肝疼`的朋友🍻 
<br/>
本仓库为`Auto Liver`的重制版
<br/>
本仓库添加了更方便使用的GUI界面，让用户可以更方便的进行`每日签到`，`获取游戏奖励`等操作
<br/>
本工具基于`Python 3.10`和`Tkinter`编写
